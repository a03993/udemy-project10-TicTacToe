## Installation

To get started with this project, simply clone the repository to your local machine:

```bash
# Clone the repository
git clone https://github.com/a03993/udemy-project10-TicTacToe.git

# Navigate to the project directory
cd udemy-project10-TicTacToe

# Install dependencies
npm install

# Start the local development server
npm run dev

# Open the file locally (if needed)
open index.html
```

## How to Play

- The game is played on a 3x3 grid.
- Players take turns marking a square with their symbol: X or O.
- The first player to align three of their symbols horizontally, vertically, or diagonally wins.
- If all nine squares are filled without a winner, the game ends in a draw.

## Features
- Intuitive and responsive user interface.
- Multiplayer mode: Play with a friend locally.
- Clear visual indicators for the current player.
- Game reset button for quick restarts.
- Tracks win and draw counts for competitive fun.
